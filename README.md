# Mina zkApp: 06-zkLedger

An attempt to build on-chain && off-chain Ledger to secure wallets with a single MINA account.

## How to build

```sh
npm run build
```

## How to build & run

```sh
npm run build && node build/src/main.js
```


## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
