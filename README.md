# Mina zkApp: 06 Ledger

This template uses TypeScript.

## How to build

```sh
npm run build
```

## How to build & run

```sh
npm run build && node build/src/main.js
```


## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
